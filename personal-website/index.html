Personal Website on AWS

This project is designed to deploy a static website involves not just creating HTML, CSS, and JavaScript files but also leveraging cloud-based tools to enhance the speed, reliability, and efficiency of deployments. The cloud used in this project is Amazon Web Services (AWS), a leading cloud services provider, offers a robust suite of tools that makes deploying, updating, and distributing static websites straightforward. By using Amazon S3, a highly-scalable object storage service, developers can host their static website assets with ease. But the true magic comes in with the integration of AWS CodePipeline, a continuous integration and continuous deployment (CI/CD) service. With CodePipeline, every code update can be automatically deployed, ensuring that the latest version of a website is always available to users. This not only removes manual deployment processes but also significantly reduces the possibility of human error. This guide offers a step-by-step overview of harnessing the power of AWS for building and deploying static websites.
